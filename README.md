# ?? Developer Octopus

Extensão para o Visual Studio Code que permite a criação de **grupos personalizados** para organizar seus arquivos de forma mais eficiente durante o desenvolvimento.

## ? Funcionalidades

- Criação de grupos (abas personalizadas)
- Interface moderna com inputs animados
- Escolha de cores para cada grupo
- Interação entre a Webview e a extensão
- Botão "Salvar" que só é habilitado após criação de um grupo
